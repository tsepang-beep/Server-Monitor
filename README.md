# Server Performance Monitoring Script

This project contains a Bash script that analyzes basic Linux server performance statistics.

## 📁 Project Structure

server-stats/
- server-stats.sh → Main script

## ⚙️ Requirements

- Linux OS (Tested on AlmaLinux)
- Bash shell
- Basic system utilities (top, ps, df, free)

## 🚀 Getting Started

Clone the repository:

git clone https://github.com/tsepang-beep/linux-automation.git

Navigate to the project:

cd linux-automation/server-stats

## ▶️ Usage

Make script executable:

chmod +x server-stats.sh

Run the script:

./server-stats.sh

## 📊 Features

- CPU usage
- Memory usage (used vs free)
- Disk usage
- Top 5 processes by CPU
- Top 5 processes by memory

## ⭐ Extra Features

- System uptime
- OS version
- Logged-in users

## 📝 Notes

- Script uses standard Linux commands
- No external dependencies required
- Works on most Linux distributions
